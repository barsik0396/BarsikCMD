Документация BarsikCMD

BarsikCMD — это CLI-утилита, которая помогает вам делать то, что нужно, быстро и без лишних движений.

📖 Это локальная документация. Актуальная версия будет доступна на основном сайте.

Установка

Добавьте здесь инструкции по установке BarsikCMD.

bash 
# Пример команды установки
barsik install

Быстрый старт

После установки вы можете сразу начать работу:

bash 
barsik --help

Основные команды

Заполните этот раздел командами BarsikCMD.

barsik help
Показать справку
barsik version
Показать версию
barsik run
Запустить утилиту

Расширенные команды

Расширенные команды для опытных пользователей. Заполните по мере необходимости.

Флаги

--help
Показать справку
--version
Показать версию
--verbose
Подробный вывод
