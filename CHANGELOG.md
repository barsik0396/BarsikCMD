# Changelog 🐾

История изменений **BarsikCMD**.  
Версионирование: **Год.Месяц.Патч** — например, `2026.3.4`.  
Новые версии — сверху. Старые записи не удаляются.

---

## 2026.3.4 — Big update

### + Добавлено
- Команда `info` — информация о системе и BarsikCMD.
- Команда `meow` — попробуй сам 🐾
- Аргумент `-help` — выводит список доступных аргументов при запуске.
- Аргумент `-disable-version-check-on-init` — отключает проверку обновлений при запуске.
- Аргумент `-load-type-select` — заглушка, будет реализован позже.
- Проверка обновлений при запуске: если доступно обновление — BarsikCMD сообщит об этом.
- 2 пасхалки. Ищи сам 😏
- Большие команды (`update`, `meow`) вынесены в отдельную папку `commands`.
- `.deb`-пакет теперь подписывается.

### ~ Изменено
- `Ctrl+C` теперь не завершает BarsikCMD, а выводит подсказку тёмно-жёлтым цветом:  
  `Для выхода введи 'exit' или нажми Ctrl+D.`

### ✗ Исправлено
- `Ctrl+\` вызывал вылет BarsikCMD — исправлено.

---

## 2026.3.3 — Update fix

### ✗ Исправлено
- Команда `update` больше не сообщает о доступном обновлении, если его нет.
- Улучшена стабильность проверки версии.

---

## 2026.3.2 — First update

### + Добавлено
- Команда `update` для автоматического обновления BarsikCMD.
- Цветной вывод в терминале.

### ~ Изменено
- Улучшена читаемость вывода команд.

### ⚠ Известные проблемы
- Команда `update` иногда ошибочно определяла наличие обновления → исправлено в 2026.3.3.

---

## 2026.3.1 — First release

### + Добавлено
- Первый официальный релиз BarsikCMD.
- Базовые команды: `clear`, `echo`, `exit`, `help`, `ver`.
- Базовый интерфейс командной строки.
- Поддержка Debian-based Linux.
