# Contributing to BarsikCMD 🐾

Любой вклад важен — от исправления опечатки до новой команды. Спасибо, что хотите помочь.

---

## Как начать

1. Сделайте **fork** репозитория.
2. Создайте новую ветку:
   ```bash
   git checkout -b my-change-name
   ```
3. Внесите изменения.
4. Убедитесь, что проект собирается и базовые команды работают.
5. Откройте **Pull Request** с понятным описанием.

---

## Pull Requests

Чтобы PR было проще принять:

- Делайте изменения логичными и не слишком объёмными.
- В описании укажите: что изменено, зачем, есть ли побочные эффекты.
- Если PR исправляет баг — опишите, как его воспроизвести.
- Если PR добавляет новую команду — обновите README или Wiki.

PR, которые усложняют BarsikCMD без реальной пользы, могут быть отклонены.

---

## Коммиты

Специального формата нет. Важна **понятность**:

- Заголовок — коротко и по делу: `Исправлен баг с update`, `Добавлена команда echo`.
- В теле коммита можно расписать детали: что изменено, почему, какие нюансы.

Цель — чтобы по истории было понятно, что происходило с проектом.

---

## Стиль кода

BarsikCMD написан на **C и C++**.

- Следуйте существующему стилю в проекте.
- Избегайте лишних зависимостей.
- Пишите простой и читаемый код.
- Не добавляйте «магии» там, где можно сделать явно.

BarsikCMD ценит минимализм и предсказуемость.

---

## Тестирование

Перед отправкой PR:

- Убедитесь, что проект компилируется.
- Проверьте, что базовые команды работают: `help`, `ver`, `echo`, `update`, `clear`, `exit`.
- Если исправляете баг — убедитесь, что он действительно исчез.

---

## Issues

Используйте Issues для:

- сообщений об ошибках,
- предложений новых функций,
- вопросов и идей.

Перед созданием Issue проверьте, нет ли уже похожей темы. Описывайте проблему максимально понятно.

---

## Что не приветствуется

- Добавление телеметрии или скрытого сбора данных.
- Изменения, оставляющие мусор в системе.
- Чрезмерное усложнение кода без реальной необходимости.
- PR без описания.

---

## 🐾

BarsikCMD вырос из эксперимента в полноценный инструмент. Ваш вклад помогает ему становиться лучше.
