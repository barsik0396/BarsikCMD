barsikcmd Основной инструмент BarsikCMD tool ▶
Описание

Это основной инструмент BarsikCMD — он сам. Точка входа для всей командной строки.

Запуск
$barsikcmd
barsikcmd-docs Локальная документация tool ▶
Описание

Инструмент для просмотра локальной документации BarsikCMD прямо в терминале.

Запуск
$barsikcmd-docs
