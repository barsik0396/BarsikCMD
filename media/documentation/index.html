BarsikCMD

Кастомная командная строка для Linux. Быстрая, минималистичная, твоя.

barsik — ~
BarsikCMD> ver
BarsikCMD версия 2026.3.2
BarsikCMD> echo Привет, Linux!
Привет, Linux!
BarsikCMD>
Справочник команд
clear builtin Очистить экран ▾

Очищает содержимое терминала. Встроенный аналог системного clear.

clear
Вывод
BarsikCMD> clear // вывод отсутствует — экран очищается
echo I/O Вывод текста ▾

Выводит переданный текст в стандартный вывод белым цветом.

echo [текст]
АргументОписание
[текст]Произвольная строка для вывода. Пробелы сохраняются.
Вывод
BarsikCMD> echo Привет, мир! Привет, мир!
BarsikCMD> echo  
exit system Завершить сессию ▾

Завершает текущий сеанс BarsikCMD и закрывает приложение.

exit
Вывод
BarsikCMD> exit Пока!
help info Список команд ▾

Выводит список всех доступных команд BarsikCMD.

help
Вывод
BarsikCMD> help BarsikCMD — доступные команды:
─────────────────────────────────
  clear — очистить экран
  echo — вывести текст на экран
  exit — выйти из BarsikCMD
  help — показать список команд
  update — проверить и установить обновления
  ver — показать версию BarsikCMD
─────────────────────────────────
update system Проверить обновления ▾

Проверяет наличие новой версии BarsikCMD через сервер обновлений и устанавливает её при наличии. Требует интернет-соединения и пароля для установки пакета.

update
Вывод — обновление доступно
BarsikCMD> update Проверка обновлений...
Доступна новая версия: 2026.4.1
Скачиваю пакет...
Размер пакета: 21 КБ
Устанавливаю пакет (потребуется пароль)...
BarsikCMD успешно обновлён до версии 2026.4.1!
Вывод — обновлений нет
BarsikCMD> update Обновлений нет. У вас актуальная версия (2026.3.2).
Вывод — ошибка сети
BarsikCMD> update Проверка обновлений...
Ошибка: не удалось получить информацию об обновлениях.
Канал обновлений: preview. Пакет скачивается во временную папку /tmp/ и устанавливается через pkexec dpkg.
ver info Версия BarsikCMD ▾

Выводит текущую версию BarsikCMD.

ver
Вывод
BarsikCMD> ver BarsikCMD версия 2026.3.2
